Mill.jl (Multiple Instance Learning Library) is a library built on top of Flux.jl aimed to flexibly prototype hierarchical multiple instance learning models as described in Simon Mandlik, Matej Racinsky, Viliam Lisy, Tomas Pevny (2021), Tomáš Pevný, Petr Somol (2017) and Tomáš Pevný, Petr Somol (2016). It is developed to be:

  • flexible and versatile
  • as general as possible
  • fast
  • and dependent on only handful of other packages

Watch our introductory talk from JuliaCon 2021.

Installation

Run the following in REPL:

] add Mill

Julia v1.6 or later is required.
