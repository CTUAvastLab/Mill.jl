Mill.string_start_code — Function
Mill.string_start_code()

Returns the current value of the string_start_code parameter used as a code point of the abstract string-start character. The default value of the parameter is 0x02, which corresponds to the STX character in ASCII encoding. See the Strings section of the docs for more information.
